# dan-upgrade

## Purpose / 用途
Maintain incremental updates, source mapping, and dual-host skill distribution.

维护 skillkit 的增量更新、来源映射和双宿主分发。

## When To Use / 触发场景
- Update this research skillkit
- Refresh packs after new source material arrives
- 更新这个 skillkit
- 刷新来源映射和技能布局

## Working Style / 工作方式
1. Clarify the user's research stage, target output, and current bottleneck.
2. Route the request through the pack-specific framework instead of giving generic motivation.
3. Produce a practical next step: checklist, outline, matrix, workflow, or decision note.
4. Recommend one adjacent `dan-*` skill when the task crosses boundaries.

## Scope In / 负责范围
- source refresh
- pack regeneration
- dual-host build
- coverage checks

## Scope Out / 不负责范围
- subject-matter answering for a specific research problem
- paper drafting
- private-source publication decisions

## Knowledge Pack / 对应知识包
- Read `../../knowledge/packs/upgrade/README.md` when you need the pack summary, frameworks, and checklists.

## Related Skills / 关联技能
- `dan-research`
- `dan-method-atlas`
- `dan-academic-tools`

## Output Contract / 输出要求
- Keep answers concrete and stage-aware.
- Prefer structured notes, checklists, and decision matrices over abstract encouragement.
- If the user needs another `dan-*` skill, say so explicitly and explain why.
